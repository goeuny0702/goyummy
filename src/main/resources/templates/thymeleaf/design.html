WireRobot

결과 구동, 연동 테스트 완료.(한국생산기술연구원)

설계 개념 불규칙한 지형에서 원활한 착륙을 위한 자세제어 장치 시험장비 개발

결과 한국항공대학교 납품.

OBLIQUE RING 측정기

설계 개념 DD Motor 모델별 측정가능
제작 16 Point 초정밀 GapSensor사용. 정밀 위치조절 Unit 개발.(0.005mm) DD Motor + Master Shaft 조립 후 측정 동심 및 평행도 측정

전 방향 이동 플랫폼 시스템

설계 개념 VR시뮬레이션 사용자 이동구현 장치

결과 DD Motor 개발사 측정기로 사용.(Justek)

MR, TR Slip ring 내구성 검사 장비

설계 개념 헬기의 MR, TR Slip Ring 회전 시 피로도 측정
시간, 속도 가변가능
제작 가공품 정밀가공, 조립(동심:0.01) 제원 : 1200 X 8500 X 1200   |   무게 : 260Kg
결과 DD Motor 개발사 측정기로 사용.(Justek)

BLU 도강판 타공 장비개발

설계 개념 BLU 도강판 금형에 정밀, 고속으로 타공
Spec : 타공간 거리 : 0.01X0.01mm 타공 속도 : 초당 50회
제작 3축 Motor, 정밀가공, 조립 제원 : 950 X 860 X 1320   |   무게 : 320Kg
결과 Spec 타공속도 초당 47회. 제품생산에 사용(동원기술)

기계장비 진동평가용 실험장치

결과 가공부하 및 진동 실험에 사용(한국기계연구원)

OLED IVL 휘도 검사장비

설계 개념 OLED 생산 시 패턴별 휘도 검사

제원 : 1600 X 1000 X 1700   |   무게 : 350Kg
결과 고객사 테스트 완료 후 LGD납품(LMS주식회사)
